# Promptica V2

AI-powered Prompt Engineering Optimization System

---

## 🚀 Overview

Promptica V2 is a full-stack web application that helps users improve and optimize their prompts for AI models by rewriting and enhancing them using structured rules.

---

## 🎯 Problem

Most users struggle to write effective prompts for AI systems, leading to low-quality responses.

---

## 💡 Solution

Promptica V2 improves prompts by applying structured rewriting techniques and returning a clearer, more effective version.

---

## ⚙️ Features

- Prompt input interface
- Prompt optimization engine
- Improved prompt output
- Prompt history storage
- Simple and clean UI

---
